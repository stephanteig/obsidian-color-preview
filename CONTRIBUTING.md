# Contributing to obsidian-color-preview

Thanks for your interest! Bug reports and feature requests are the most valuable contributions — please open an issue using the templates.

## Development setup

1. Clone the repo into `<vault>/.obsidian/plugins/obsidian-color-preview/`
2. `npm install`
3. `npm run dev` — esbuild watches and rebuilds `main.js`
4. Reload Obsidian (or use the Hot Reload plugin) to test changes

## Pull requests

- Open an issue first for anything beyond a small fix, so we agree on the approach
- Branch from `main`, use a descriptive kebab-case branch name
- Run `npm run build` and test in a real vault before submitting
- Keep PRs focused — one change per PR
