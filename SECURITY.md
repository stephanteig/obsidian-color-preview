# Security Policy

## Supported versions

Only the latest release of obsidian-color-preview is supported.

## Reporting a vulnerability

If you find a security issue (e.g. anything that could execute code, leak vault contents, or make network requests without consent), please **do not open a public issue**. Email **teig.stephan@gmail.com** instead. You'll get a response within a week, and a fix will be prioritized before public disclosure.
